Scatter Plot

This implementation was done using Elastic search.
Once the elastic search is up and running on the ufo.usc.edu website , please uncomment the "< iframe >" for Scatter plot in "team3_index.html"
Also change the url for the elastic search in "js/team3_js_files/scatter.js" file

This is a scatter plot for UFO sightings based on distance of closest airports from the sighting.

Here, the x-azis is the nearest large airport and y-axis is nearest medium sized airport from the UFO sightings.
So, the sightings which are at reasonably equal distances from largest and medium sized airport would be clustered together.

Donut Chart

An interesting visualization of UFO sightings over the months.
Here the number in the center represents the total UFO sightings. Whereas it shows the sightings of a given month when we hover over that segment on the donut.

Pie Chart

This pie chart shows the distribution of UFO sightings day-wise.
There is almost uniform distribution over the days, but Saturday has the most UFO sightings.

Bubble Chart

This clickable Bubble Chart is an apt representation of the number of UFO sightings based on the classes obtained from Tika Vision API.
Every bubble represents a class and has the number of UFO sightings corresponding to that class on a sampled dataset.
