HOW WELL DID OCR PERFORM IN EXTRACTING TARGETED FIELDS?

WHAT ARE WE ABLE TO ANSWER WITH THIS VISUALIZATION?

We seek to answer the question "How Well Did OCR Perform in Extracting Targeted Fields?". In order to convert the UK Ministry of Defense UFO sighting PDFs to usable data, OCR was employed to extract the content of the reports. To best show the performance metrics achieved by OCR, we decided to plot a Weighted Tree showing the number of fields OCR was able to extract as well as the number of valid and invalid values returned.


By looking at the Weighted Tree, we can show specifically where the tool struggled as well as which areas it excelled in extracting data.
